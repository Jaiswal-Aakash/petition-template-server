# Template Structure

## Template 1: Petition under Section 355(1) - Absence Condonation

Based on the provided PDF (sc 325 Aswin sanjeev 355.pdf), this template includes:

- Court header
- Case number
- Petitioner/Accused names
- Respondent/Complainant information
- Section 355(1) reference
- Petition points (4 standard points)
- Prayer section
- Date and place
- Signature section
- Address for service section

### Template Variables:

- `{{courtName}}` - Court name (e.g., "COIMBATORE")
- `{{caseNumber}}` - Case number (e.g., "S.C.No. 325 of 2025")
- `{{petitionerNames}}` - Accused names (e.g., "SANJEEVKUMAR – A2, ASWINKUMAR – A3")
- `{{respondentName}}` - Respondent name (e.g., "INSPECTOR OF POLICE")
- `{{respondentAddress}}` - Respondent address
- `{{sectionNumber}}` - Section number (e.g., "355(1)")
- `{{point1}}`, `{{point2}}`, `{{point3}}`, `{{point4}}` - Petition points
- `{{prayerText}}` - Prayer text
- `{{day}}`, `{{month}}`, `{{year}}` - Date components
- `{{place}}` - Place name
- `{{lawyerSignaturePath}}` - Signature image (base64 data URI)
- `{{advocateName}}`, `{{advocateAddress}}`, `{{advocatePhone}}` - Advocate details
